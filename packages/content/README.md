# packages/content

Curated verse library, prompt pool, and LLM movement templates. **Phase 1 — not yet scaffolded.**

When this is built, it will be:
- A shared TypeScript package consumed by `apps/mobile`, `apps/web`, and `apps/api`
- Source: public-domain translations (KJV, Reina-Valera 1960, Louis Segond 1910, Chinese Union Version, etc.)
- Locale-aware: every verse has translations in all shipped locales
- The LLM prompt templates are versioned here

See `docs/LOCALIZATION.md` for the localization plan.
